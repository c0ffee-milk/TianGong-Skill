# 天工.skill

> **助手蒸馏 x 对抗攻防？GAN 驱动的高纯度思维蒸馏引擎**

> **天工来帮你锤炼任何一个Skill**
>

<br> 大多数"角色扮演 Skill"停在表层——得到的是模仿，不是蒸馏。
我们再向前走一步：**生成器**先从海量语料中提炼思维晶核，**判别器**随测试纯度并将缺陷提交给生成器，反复迭代，直到 Skill 能在压力下稳定输出目标人物高纯度表达。<br>

输入一个名字，天工自动完成调研 → 对抗提炼 → 验证全流程，交付一个可运行的高纯度人格Skill。
